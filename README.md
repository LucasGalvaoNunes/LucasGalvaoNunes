## 👨‍💻 About Me

I'm a **Senior Software Engineer** with over 8 years of experience, specializing in backend architecture, distributed systems, and highly critical environments. Currently, I'm building high-availability, scalable fraud prevention solutions at **Neon** using Go, .NET, Kafka, and microservices.

My journey started in 2013 with game development, and today I focus on crafting robust event-driven architectures and clean code that makes a real-world impact.

* 🔭 **Currently building:** My Open Source Lab — creating free VS Code extensions, finance apps, and everyday dev tools.
* 🌱 **Currently exploring:** Advanced observability, robust data pipelines, and pushing the boundaries of software engineering applied to digital businesses.
* ⚡ **Outside of code:** You can usually find me dialing in my audio interface to my Edifier R1280t monitors for a Rocksmith guitar session, or recording my best missed shots as my CS2 parody persona, "Calvaocs," on Instagram.
